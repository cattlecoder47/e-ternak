

## About E-Ternak

Aplikasi digital yang berfokus memberdayakan peternak


**Installation**
1. Install dependency composer
   `composer update`
2. Setup .env <br>
   `DB_CONNECTION=yourdatabasedriver`<br>
   `DB_HOST=yourdatabasehost`<br>
   `DB_USERNAME=yourdatabaseuser`<br>
   `DB_PASSWORD=yourdatabasepassword`<br>
   `DB_DATABASE=db_eternak`<br>
